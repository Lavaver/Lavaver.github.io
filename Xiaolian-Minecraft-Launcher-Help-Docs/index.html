说明

本帮助文档皆在帮助各位玩家在游玩时的一些问题。

如果遇到此处没有的问题，请联系我们。

常见问题

此处列举出来了在使用该启动器的常见问题。

没有 MSVCP140.dll 库文件

错误原因：

电脑内没有安装官方提供的 Visual C++ Redistributable for Visual Studio 2015 补丁包。

解决方法：

点击此处，按照提示选择与系统位数匹配的版本（64位的选x64，32位的选x86）

下载完成打开程序，等待程序自动安装完毕。

若你正在运行启动器，在安装结束后你需要重启启动器。

系统找不到相应文件

错误原因：

由于该启动器依赖于 CMCL 核心，当不存在该文件时会报错。

解决方法：

重新下载 XMCL 完整包。

下载游戏时报错

错误原因：

由于文件尚未上传到服务器，且不存在该路径时发生。

解决方法：

更新启动器以暂时禁用该功能（直至声明恢复功能）。

无法快速启动游戏

错误原因：

由于内核的原因，导致游戏并未按照预期启动。

解决方法：

直接忽略，点击按钮继续游戏。

使用方法

  • 进入主界面后，在右侧栏找到“设置”选项卡

  • 在内存分配一栏中填写为游戏分配的最大内存（不能是其他字符，只能是数字）

  • 点击“分配内存”按钮，继续后等待提示分配完成

  • 回到主界面，直接启动游戏

  • 开玩啦！
